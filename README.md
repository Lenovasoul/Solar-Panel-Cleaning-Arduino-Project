# Solar-Panel-Cleaning-Arduino-Project
A basic idea for telling people to clean off Solar Panels

This program takes in a file and then based on the inputs gained in the file it'll give an output to an RGB LED

Required Libraries:
MATLAB Supprot Package for Arduino Hardware, Arduino LCD Connection

Required Hardware:
Arduino - UNO R3, USB Cable, 1k or 330 Resistor, 4 Breadboard Jumper Cables, A RGB LED, and a Breadboard // The smaller Breadbaord that comes in the kit works well
